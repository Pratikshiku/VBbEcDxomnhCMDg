## Library Seat Reservation Management System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/4dd589e3bbfa4132b220ab0a4ea84553/ghb20250919) 

### 1. Related Technologies

- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript`, etc.
- Database: `MySQL`

### 2. Related Software (any of the listed software can be used)

- `IDEA`
- `Eclipse`
- `Navicat`
- `Visual Studio Code(VScode)`
- etc.

### 3. Functional Description

The system is divided into (roles): users, administrators.

**User Functions:**

1. Login
2. Register
3. Announcement Display
4. View Seats
5. Reserve Seats
6. View Reservations
7. Personal Center
8. Change Password


**Administrator Functions:**

1. Login
2. Publish Announcements
3. Publish Seats
4. View User Reservations
5. Manage Users
6. Change Password
